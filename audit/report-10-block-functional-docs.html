B01: Hero Banner

P0 - Critical EDS: hero 5 Variants 8 Templates
Full-width hero section at the top of pages featuring a large background image or video with overlaid text content, heading, and call-to-action.

Variant A: Image Background + White Text Overlay

HeroP0

Standard hero with a full-width background image and white text overlaid directly on the image with a dark overlay or positioned on a darker region.

Structure / Fields

FieldTypeRequiredDescription
Background ImageImageRequiredFull-width background image from DAM
H1 HeadingTextRequiredPrimary heading, F37 Lineca Medium, 64px desktop
DescriptionRich TextOptionalSubtitle paragraph, Roboto Regular 18px
CTA LinkLinkOptionalCall-to-action link/button

Acceptance Criteria

  • Background image fills 100% viewport width
  • Image scales proportionally (object-fit: cover)
  • Image maintains minimum height of 416px on desktop
  • Text color: white (#FFFFFF) on dark backgrounds
  • Sufficient contrast with white text ensured

Interactions

Hover
CTA link underlines or shows arrow animation
Focus
Visible focus ring on CTA link

Responsive Behavior

Desktop

1440px+

  • Full-width image with text overlay
  • H1: 64px / 68px line-height
  • Min-height: 416px
Tablet

768px - 1199px

  • Image displayed, text may move below
  • H1: 40px / 44px line-height
Mobile

< 768px

  • Image stacked above text
  • H1: 30px / 34px line-height
  • Reduced spacing

Variant E: Image + Anchor Link

HeroP0

Similar to Variant A but the CTA is an in-page anchor link that scrolls to the next section rather than navigating away.

Acceptance Criteria
